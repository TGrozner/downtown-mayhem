# VFX Flipbook Atlases

The `unity-*.webp` files in this folder are converted runtime atlases from
Unity Labs' free VFX Image Sequences and Flipbooks release.

Source:
https://unity.com/blog/engine-platform/free-vfx-image-sequences-flipbooks

Upstream asset names:
- `Explosion01_5x5.tga`
- `Explosion01-nofire_5x5.tga`
- `FireBall02_8x8.tga`
- `WispySmoke01_8x8.tga`

Unity released these image sequences under CC0. The original TGA flipbooks were
converted to WebP for browser delivery with ffmpeg.
